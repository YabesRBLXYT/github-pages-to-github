GitHub Pages to GitHub Repository Converter

Masukkan link GitHub Pages:

Catatan: Tool ini tidak mendukung link dengan lanjutannya (misalnya /something di akhir link).

Tutorial 1: Cara Membuat GitHub Pages

  1. Buka repository GitHub yang ingin Anda gunakan untuk GitHub Pages.
  2. Pilih tab Settings di repository tersebut.
  3. Scroll ke bawah hingga menemukan section GitHub Pages.
  4. Setel Source menjadi main branch atau branch lainnya sesuai kebutuhan.
  5. Setelah diaktifkan, GitHub Pages Anda akan tersedia di link dengan format: https://username.github.io/repository.

Tutorial 2: Cara Convert GitHub Repository ke GitHub Pages

Sebelum melanjutkan, silakan kunjungi tools pertama untuk memulai proses konversi GitHub repository ke GitHub Pages.

  1. Salin link GitHub repository yang ingin di-convert (contoh: https://github.com/username/repository). Pastikan bahwa repository tersebut sudah memiliki GitHub Pages yang diaktifkan.
  2. Buka tools pertama dan masukkan link tersebut untuk mendapatkan link GitHub Pages yang sesuai dengan repository Anda.
  3. Klik tombol Convert, dan Anda akan mendapatkan link GitHub Pages yang sudah siap digunakan.
  4. Anda bisa menyalin atau mengunjungi link tersebut untuk memastikan bahwa GitHub Pages berfungsi dengan baik.

Tutorial 3: Cara Convert GitHub Pages ke GitHub Repository

  1. Salin link GitHub Pages yang ingin Anda konversi (contoh: https://username.github.io/repository). Pastikan link tersebut mengarah ke GitHub Pages yang valid.
  2. Buka halaman ini dan masukkan link GitHub Pages yang telah Anda salin ke dalam kotak input di atas.
  3. Klik tombol Convert. Jika link valid, link GitHub repository asli akan muncul di bawah kotak input.
  4. Salin atau klik link tersebut untuk membuka GitHub repository asli di tab baru.
